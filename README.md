# Rahul Pamula - Tailored Resumes

## 🛠 Compilation

To compile these `.tex` files into PDFs, you will need a LaTeX distribution installed on your system (such as [TeX Live](https://www.tug.org/texlive/), [MiKTeX](https://miktex.org/), or [MacTeX](https://www.tug.org/mactex/)). 

Alternatively, you can upload them to web-based LaTeX editors like [Overleaf](https://www.overleaf.com/) to render them easily.

### Local Command Line Example:
```bash
pdflatex rahul_fullstack.tex
```
